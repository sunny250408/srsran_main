#srsRAN Tutorial Project

srsRAN을 기반으로 한 srsUE를 사용한 srsRAN gNB

-----

##환경 정보
-OS : Ubuntu 22.04
-srsRAN: v22.10
-Core : Open5GS
-연결 방식 : ZMQ(소프트웨어 기반 연결)

-----

##폴더 구성
- `libzmq/` - ZMQ 라이브러리
- `srsRAN_Project/` - gNB 관련 설정 및 실행 코드
- `srsRAN_4G/` - UE 관련 설정
- `uhd/` - SDR 드라이버 구성 파일
